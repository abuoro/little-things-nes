Insane Game
===========
Very unfinished; no scoring, no gravity.

Title Screen
------------
At the title screen, press B to start a new game in puzzle mode
or A to start a new game in action mode.
Press Up to start the virtual trumpet.

Trumpet
-------
Select, Start, and B are the "valves";
pressing various directions on the Control Pad blows into trumpet.

Virtual trumpet will be one of the easter eggs in the final release.
